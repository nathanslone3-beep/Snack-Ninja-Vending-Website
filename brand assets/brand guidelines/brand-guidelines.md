# Snack Ninja Vending — Brand Guidelines

> Source: snackninjavending.com | Last reviewed: May 2026

---

## 1. Brand Overview

**Company:** Snack Ninja Vending  
**Industry:** AI-powered vending / smart amenities  
**Market:** Commercial property operators — offices, gyms, schools, apartments  
**Positioning:** A tech-forward vending partner that installs, stocks, and manages modern AI smart coolers with zero upfront cost and zero disruption to the client.

---

## 2. Brand Personality

The Snack Ninja identity lives at the intersection of **stealth efficiency** and **high-tech precision**. The brand is confident and contemporary with a playful edge — the "ninja" metaphor signals silent, seamless operation rather than gimmickry.

| Trait | What it means in practice |
|---|---|
| Stealthy | Operates invisibly; clients never worry about restocking |
| Tech-forward | AI inventory management, cashless, smart cooler UX |
| Premium | Elevated snack selection, sleek machine design |
| Approachable | No jargon-heavy sales pitch; direct, human tone |
| Dependable | Hands-off for clients; "set it and forget it" reliability |

---

## 3. Brand Voice & Tone

### Core Taglines
- **"Simple. Stealthy. Seamless."** — primary tagline (alliterative, punchy, memorable)
- **"Modern Amenities Made Simple"** — secondary/explanatory tagline

### Voice Principles
- **Action-oriented:** Lead with verbs and outcomes. ("Upgrade your space." not "We offer upgrade services.")
- **Minimal friction:** Short sentences. No jargon.
- **Confident, not arrogant:** State capability without overselling.
- **Ninja language used sparingly:** The metaphor should feel intentional, not forced.

### Power Words (use frequently)
`silent` · `lightning-fast` · `hands-off` · `seamless` · `premium` · `streamlined` · `smart` · `cashless` · `zero disruption` · `zero upfront cost` · `AI-powered` · `modern`

### Words to Avoid
- Legacy vending language: "machine," "coin-op," "snack box"
- Passive voice constructions
- Overuse of "ninja" as a filler adjective

### Sample Copy Patterns
```
Headline: [Bold outcome]. [One supporting detail].
Body: [What we do] + [how it benefits you] + [what you don't have to do].
CTA: Learn more  |  Request a demo  |  Get started
```

## 5. Color Palette

The website employs a **dark, tech-forward aesthetic** described internally as a "502 tron theme" — deep backgrounds with high-contrast text and neon-inspired accents. The palette below reflects the visual direction; verify exact hex values against the live site's CSS when applying to new assets.

### Primary Colors

| Name | Hex | Usage |
|---|---|---|
| Ninja Black | `#0D0D0D` | Primary background, large surfaces |
| Midnight Dark | `#1A1A2E` | Section backgrounds, card fills |
| Pure White | `#FFFFFF` | Primary body text on dark backgrounds |

### Accent Colors (Tron / Tech Theme)

| Name | Hex | Usage |
|---|---|---|
| Neon Cyan | `#00E5FF` | Primary CTA buttons, links, highlight borders |
| Electric Green | `#39FF14` | Secondary accent, success states, icons |
| Muted Steel | `#8892A4` | Secondary text, captions, metadata |

### Neutral / Utility

| Name | Hex | Usage |
|---|---|---|
| Off-White | `#F0F0F0` | Text on mid-tone backgrounds |
| Dark Gray | `#2C2C3E` | Dividers, card strokes, input borders |

> **Note:** These color values are derived from the site's described visual aesthetic. Confirm against actual CSS source if implementing production assets.

### Color Do's & Don'ts
- **Do** pair neon accents against dark backgrounds for maximum contrast
- **Do** use white as the default text color on all dark sections
- **Don't** use neon-on-neon combinations
- **Don't** use more than two accent colors in a single layout

## 6. Typography

The site uses clean, modern sans-serif typography consistent with the Squarespace platform defaults and a tech-forward brand.

### Type Scale

| Role | Style | Weight | Notes |
|---|---|---|---|
| Hero Headline | Sans-serif | ExtraBold / Black | Large, impact-first |
| Section Heading | Sans-serif | Bold | Clear hierarchy |
| Subheading | Sans-serif | SemiBold | Supporting context |
| Body Copy | Sans-serif | Regular | Readable, generous leading |
| CTA / Button | Sans-serif | Bold | All-caps or Title Case |
| Caption / Meta | Sans-serif | Light or Regular | Muted color, smaller size |

### Typography Rules
- Headlines should feel **bold and commanding** — this is not a subtle brand
- Body text line-height should be at least 1.6 for readability
- Avoid decorative or script fonts — they conflict with the tech identity
- Underscores or emphasis markers in headings should be used purposefully, not decoratively

## 7. Imagery & Photography

### Style Direction
- **Product-first:** AI smart coolers (Area 502 units) are the hero subject
- **Clean & high-contrast:** White or dark studio-style backgrounds preferred
- **Tech context:** Phone mockups showing the cooler interface reinforce the AI angle
- **Real environments:** Offices, gyms, schools — show coolers in situ, not isolated

### Image Types in Use
| Type | Usage |
|---|---|
| Product photography (front/side views) | Primary product pages, hero sections |
| Phone mockups (app/UI screen) | Process and feature sections |
| Desktop browser screenshots | Tech/dashboard callouts |
| Brochure-style graphics | Sales materials, PDF leave-behinds |

### Imagery Principles
- **Do** use sharp, well-lit product shots that show the cooler clearly
- **Do** feature modern, clean environments that the target client operates
- **Don't** use stock photography that feels generic or unrelated to vending/facilities
- **Don't** use cluttered or low-resolution images at any size

---

## 8. UI & Layout Patterns

### Layout Principles
- **Mobile-first:** All layouts must render cleanly on phone before desktop
- **Whitespace-generous:** Breathing room signals premium quality
- **Product-centric hierarchy:** Cooler images and key stats anchor every page
- **Minimal navigation:** Home · Our AI Smart Coolers · Our Process · Services · Contact

### Button Styles
- Primary CTA: Filled, neon accent color, bold text, rounded or pill shape
- Secondary CTA: Outlined/ghost version of primary
- Standard label: "Learn more" · "Request a demo" · "Get started"

### Section Structure Pattern
```
[Hero: Bold headline + subline + CTA]
[Feature block: Icon/image + short description]
[Social proof or differentiation callout]
[Secondary CTA or contact prompt]
```

## 9. Brand Differentiators (Key Messages)

Use these as content pillars across all channels:

1. **Zero upfront cost** — clients pay nothing to get started
2. **Hands-off management** — Snack Ninja handles all stocking and servicing
3. **AI-powered inventory** — smart tech means no stockouts, no guesswork
4. **Cashless & modern** — frictionless purchase experience for end users
5. **Customizable branding** — machine wraps can match a client's space or brand
6. **Premium product selection** — elevated snack and beverage options
